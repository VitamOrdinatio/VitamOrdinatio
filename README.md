# Gary Vanderlaan

Molecular biologist, educator, and bioinformatics practitioner building reproducible computational workflows for genomics, transcriptomics, rare disease analysis, and clinical data harmonization.

My portfolio is organized as an interconnected ecosystem of bioinformatics repositories rather than a set of isolated projects. The core framework emphasizes reproducibility, modular pipeline design, and transparent documentation, while individual repositories address specific layers of analysis such as variant annotation, gene set integration, transcriptomics, database-backed storage, and rare disease prioritization. Together, these projects reflect my transition from manual, domain-rich biological analysis toward more rigorous computational architecture, with a focus on clinically relevant genomics workflows and scalable research design.

My work focuses on integrating biological insight with reproducible software pipelines using Python, Bash, SQL, and public biomedical datasets.

---

## Portfolio Repositories

### inherited_anemias
Codebase supporting the Total et al., 2025 publication. Analysis and visualization of inherited anemia datasets and clinical metrics.

### variant_annotation_pipeline
Reproducible workflow for annotating genetic variants using public databases and structured annotation pipelines.

### variant_database
Database design and data engineering project for storing, indexing, and querying variant and annotation data.

### rare_disease_gene_prioritization
Computational approaches for prioritizing candidate genes in rare disease contexts using variant, expression, and annotation data.

### rnaseq_pipeline
Reproducible RNA-seq workflow from raw data through counts, differential expression, and downstream interpretation.

### reproducible_pipeline_framework
General framework for building modular, reproducible scientific computing pipelines.

### picu_harmonization
Clinical data harmonization project focused on pediatric ICU-style laboratory metrics across heterogeneous biomedical literature sources.
